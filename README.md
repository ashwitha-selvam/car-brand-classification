# Car-Brand-Classification
Deep-learning based web application that will predict the brand of a car.

# What is this project about?
* I created this project to get started with **deep learning**. Specifically, how to predict classes using images via **Deep Learning**.
* I have used the concept of *transfer learning*. Transfer learning is a deep learning method, where a model developed previously for a task, is reused as the starting point for a model on a secondary task. More about transfer-learning, <a href="https://machinelearningmastery.com/transfer-learning-for-deep-learning/">here</a>.

# Algorithms used:
As mentioned earlier, I've used transfer learning in this project. Specifically, I've used **ResNet 50** algorithm. More about ResNet50, <a href="https://iq.opengenus.org/resnet50-architecture/#:~:text=ResNet50%20is%20a%20variant%20of,explored%20ResNet50%20architecture%20in%20depth./">here</a>.

# Dataset used:
Dataset is provided in the repository itself.

# Accuracy score:
After running the model for 30 epochs, I was able to achieve:
* **Accuracy**: 87.50%
* **Validation-accuracy**: 70.69%;
